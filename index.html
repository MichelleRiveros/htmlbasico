Etiquetas contenedoras

1. Etiqueta div

Permite agregar informacion semantica no implicita

2. Etiqueta main

main representa el contenido principal de un body, tener encuenta que esta Etiqueta no te permite agregarla dentro de otras etiquetas, pero lo que si permite es que otras esten dentro de ella, y solo se puede declarar una etiqueta main en la pagina web.

3. Etiqueta section

Es un elemento que hace referencia a cada uno de los bloques genericos de un gran contenido de la pagina web. Se pueden declarar varias de estas etiquetas.

4. Etiqueta header

Representa el contenido introductorio de la pagina web (ENCABEZADO)

5. Etiqueta nav

Representa el menu de navegacion en la pagina web

6. Etiqueta article

Representa un contenido que puede ser reutilizado independientemente
